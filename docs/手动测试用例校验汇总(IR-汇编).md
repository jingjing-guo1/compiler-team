# 编译器测试用例汇总表
## 说明
1. 测试对象：C 子集编译器（生成 x86-64 AT&T 汇编）
2. 判定规则：
   - ✅ 正常：运行结果与 C 标准预期一致
   - ⚠️ 结果巧合正确、语义实现错误

3. 分类：基础运算、分支语句、循环语句、混合嵌套、函数调用、块级作用域

## 三、循环语句（while）测试用例
### 用例10 双层 while 嵌套累加
```c
int main() {
    int i = 0;
    int j = 0;
    int sum = 0;
    while (i < 2) {
        while (j < 3) {
            sum = sum + 1;
            j = j + 1;
        }
        j = 0;
        i = i + 1;
    }
    return sum;
}
```
预期返回值：`6`
实际运行结果：`6`
测试结论：✅ 正常

### 用例11 三层 while 嵌套累加
```c
int main() {
    int a = 0, b = 0, c = 0;
    int cnt = 0;
    while (a < 2) {
        while (b < 2) {
            while (c < 2) {
                cnt = cnt + 1;
                c = c + 1;
            }
            c = 0;
            b = b + 1;
        }
        b = 0;
        a = a + 1;
    }
    return cnt;
}
```
预期返回值：`8`
实际运行结果：`8`
测试结论：✅ 正常

## 六、块级作用域专项测试用例（核心缺陷用例）
### 用例16 多层代码块 + 多层同名变量
```c
int main() {
    int val = 100;
    if (1) {
        int val = 200;
        if (1) {
            int val = 300;
            return val;
        }
    }
    return val;
}
```
预期返回值：`300`
实际运行结果：`300`
测试结论：⚠️ 结果巧合正确，**语义错误**
说明：所有同名变量共用同一栈地址，变量互相覆盖，未实现块级隔离。
