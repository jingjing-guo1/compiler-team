# 三地址码生成器实现说明

## 成员5任务完成情况

### 1. 实现概述

已完成语法树到三地址码（Three-Address Code）的翻译功能，实现了以下核心功能：

- ✅ 使用成员1定义的四元式结构体（IRQuad）和中间代码结构（IRCode）
- ✅ 遍历语法树，按规则生成代码
- ✅ 表达式处理：递归生成左右子树的代码，产生临时变量
- ✅ 赋值语句：生成右部表达式代码，然后生成 ASSIGN
- ✅ if语句：生成条件代码、条件跳转、then块、else块、结束标签
- ✅ while语句：生成循环标签、条件代码、条件跳转、循环体、跳回
- ✅ 临时变量计数 new_temp()
- ✅ 数组访问支持
- ✅ break/continue语句支持
- ✅ 函数调用支持
- ✅ 逻辑运算短路求值

### 2. 核心数据结构

#### 四元式结构（来自 common.h）

```c
typedef enum {
    IR_ADD, IR_SUB, IR_MUL, IR_DIV, IR_MOD,
    IR_ASSIGN,
    IR_LT, IR_LE, IR_GT, IR_GE, IR_EQ, IR_NE,
    IR_AND, IR_OR, IR_NOT,
    IR_LABEL, IR_GOTO, IR_IF_FALSE_GOTO,
    IR_RETURN, IR_CALL,
    IR_FUNC_ENTER, IR_FUNC_EXIT
} IROp;

typedef struct IRQuad {
    IROp op;        // 操作码
    char *arg1;     // 第一个操作数
    char *arg2;     // 第二个操作数
    char *result;   // 结果
} IRQuad;
```

### 3. 主要函数实现

#### 3.1 表达式生成（gen_expr）

处理以下表达式类型：

- **NODE_NUMBER**: 数字常量
  ```
  输入: Num(10)
  输出: t1 = 10
  ```

- **NODE_LVAL**: 变量引用
  ```
  输入: Var(a)
  输出: t1 = a
  ```

- **NODE_ARRAY_ACCESS**: 数组访问
  ```
  输入: arr[index]
  输出: t1 = index, t2 = arr[t1]
  ```

- **NODE_UNARY_EXPR**: 一元表达式
  ```
  输入: -expr
  输出: t1 = expr, t2 = 0, t3 = t2 - t1
  ```

- **NODE_BINARY_EXPR**: 二元表达式
  ```
  输入: a + b
  输出: t1 = a, t2 = b, t3 = t1 + t2
  ```

- **NODE_FUNC_CALL**: 函数调用
  ```
  输入: func(a, b)
  输出: t1 = a, t2 = b, t3 = call func
  ```

#### 3.2 条件跳转生成（gen_condition）

支持短路求值的逻辑运算：

- **逻辑与（&&）**:
  ```
  if (a && b) goto true_label else goto false_label
  生成:
    if (!a) goto false_label
    L1:
    if (!b) goto false_label
    goto true_label
  ```

- **逻辑或（||）**:
  ```
  if (a || b) goto true_label else goto false_label
  生成:
    if (a) goto true_label
    L1:
    if (!b) goto false_label
    goto true_label
  ```

#### 3.3 语句生成（gen_stmt）

处理以下语句类型：

- **NODE_BLOCK**: 代码块
  ```
  依次处理块内的每个语句
  ```

- **NODE_VAR_DECL / NODE_CONST_DECL**: 变量/常量声明
  ```
  输入: int a = 10;
  输出: t1 = 10, a = t1
  ```

- **NODE_ASSIGN_STMT**: 赋值语句
  ```
  输入: a = b + c
  输出: t1 = b, t2 = c, t3 = t1 + t2, a = t3
  ```

- **NODE_IF_STMT**: if语句
  ```
  输入: if (cond) then_stmt else else_stmt
  输出:
    if (!cond) goto L_else
    then_stmt代码
    goto L_end
    L_else:
    else_stmt代码
    L_end:
  ```

- **NODE_WHILE_STMT**: while语句
  ```
  输入: while (cond) body
  输出:
    L_start:
    if (!cond) goto L_end
    body代码
    goto L_start
    L_end:
  ```

- **NODE_BREAK_STMT**: break语句
  ```
  输出: goto break_label
  ```

- **NODE_CONTINUE_STMT**: continue语句
  ```
  输出: goto continue_label
  ```

- **NODE_RETURN_STMT**: return语句
  ```
  输入: return expr
  输出: t1 = expr, return t1
  ```

### 4. 示例演示

#### 示例1：简单赋值表达式

**输入语法树**:
```
AssignStmt
  left: LVal(a)
  right: BinaryExpr('+')
    left: Num(10)
    right: LVal(b)
```

**输出三地址码**:
```
t1 = 10
t2 = b
t3 = t1 + t2
a = t3
```

#### 示例2：if语句

**输入语法树**:
```
IfStmt
  cond: BinaryExpr('<')
    left: LVal(a)
    right: Num(10)
  then: AssignStmt
    left: LVal(b)
    right: Num(1)
  else: AssignStmt
    left: LVal(b)
    right: Num(0)
```

**输出三地址码**:
```
t1 = a
t2 = 10
t3 = t1 < t2
if_false t3 goto L1
t4 = 1
b = t4
goto L2
L1:
t5 = 0
b = t5
L2:
```

#### 示例3：while循环

**输入语法树**:
```
WhileStmt
  cond: BinaryExpr('>')
    left: LVal(n)
    right: Num(1)
  body: Block
    AssignStmt: result = result * n
    AssignStmt: n = n - 1
```

**输出三地址码**:
```
L1:
t1 = n
t2 = 1
t3 = t1 > t2
if_false t3 goto L2
t4 = result
t5 = n
t6 = t4 * t5
result = t6
t7 = n
t8 = 1
t9 = t7 - t8
n = t9
goto L1
L2:
```

#### 示例4：函数调用

**输入语法树**:
```
FuncCall(factorial)
  args: LVal(x)
```

**输出三地址码**:
```
t1 = x
t2 = call factorial
```

### 6. 关键改进点

相比原始实现，主要改进：

1. **移除了 SymbolTable 参数**：简化函数签名，语义信息已在语法树中
2. **修复了条件跳转逻辑**：正确处理 if 和 while 的条件跳转
3. **添加了数组访问支持**：处理 NODE_ARRAY_ACCESS 节点
4. **添加了 break/continue 支持**：正确处理循环控制语句
5. **实现了短路求值**：逻辑与和逻辑或的短路求值
6. **移除了递归调用 sibling**：避免重复处理语句
7. **改进了错误处理**：添加了适当的错误报告

可以使用以下测试用例验证功能：

4. **函数调用**: `y = factorial(x)`
5. **数组访问**: `a[i] = a[i] + 1`

### 8. 注意事项

1. 临时变量命名：使用 t0, t1, t2... 格式
2. 标签命名：使用 L0, L1, L2... 格式
3. 数组访问：使用 arr[index] 格式表示
4. 函数调用：当前实现简化了参数传递，实际应用中可能需要扩展
5. 立即数：直接在 IR_ASSIGN 的 arg1 中存储数字字符串
